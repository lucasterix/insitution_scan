CSV-Batch-Import

Eine Institutionsliste hochladen. Alle Domains werden gescannt, die Angebote gehen morgen ab 08:00 Uhr getaktet raus.

So funktioniert's
  1. CSV mit Spalten name,email hochladen (mehrere E-Mails durch ; trennbar).
  2. Die Domain wird aus der ersten E-Mail-Adresse gezogen. Doppelte Domains werden übersprungen.
  3. Für jede Domain wird ein Scan in die Queue gestellt — läuft sofort.
  4. Nach Scan-Ende wird automatisch eine geplante Angebots-Mail angelegt: Versand morgen ab 08:00 Uhr, alle {{ spacing_minutes }} Minuten eine Mail (Spam-Schutz).
  5. Jede geplante Mail taucht im Posteingang unter „Geplante Nachrichten" auf — dort bis Versandzeit jederzeit einzeln abbrechbar.
Max. {{ max_rows }} Zeilen pro Upload. Dateigröße ≤ 2 MB.

Erste Zeile muss die Kopfzeile name,email sein.

Jede Domain bekommt einen eigenen Versand-Slot. Bei {{ spacing_minutes }} min Abstand dauert eine 30er-Liste ca. 2 Stunden vom Start bis zum letzten Versand.

Abbrechen

Hinweis: Der CSV-Import bestätigt den Scan-Umfang im Sinne unseres Arbeitsversprechens (klar abgegrenzter Prüfumfang). Die Angebots-Mails sind Erstkontakt — bitte vor dem Versand kurz im Posteingang durchgehen.
